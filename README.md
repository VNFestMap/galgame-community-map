## 📖 项目简介

本项目是一个可视化的全国 Galgame / 视觉小说高校同好会分布地图，帮助同好快速找到身边的组织。

### 主要功能

- 🗺️ **中国地图可视化** - 各省份同好会数量一目了然
- 🔍 **全局搜索** - 按组织名或群号快速查找
- 🏷️ **分类筛选** - 高校同好会 / 地区高校联合 / 视觉小说学园祭
- 📋 **一键复制** - 点击即可复制联系方式
- 🌙 **深色模式** - 支持明暗主题切换
- 📱 **移动端适配** - 手机浏览器完美支持

### 数据统计

- 覆盖 30+ 省份/地区
- 200+ 高校同好会
- 持续更新中...
