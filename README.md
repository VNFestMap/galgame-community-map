# 全国 Galgame 同好会地图

> 基于 [china-bandori-maps](https://github.com/HELPMEEADICE/china-bandori-maps) 二次开发
> 将邦邦群地图改造为全国 Galgame / 视觉小说高校同好会地图

[![许可证](https://img.shields.io/badge/License-GPLv3-blue.svg)](LICENSE)
[![Galgame](https://img.shields.io/badge/主题-Galgame-ff69b4.svg)]()

本项目是一个可视化的全国 Galgame / 视觉小说高校同好会分布地图，帮助同好快速找到身边的组织。

- 正式域名：https://www.map.vnfest.top

## 📁 项目结构

- 本网站数据来源于各高校同好会公开信息，如有错误或侵权请联系删除
- 联系方式仅供参考，请自行核实信息准确性
- 本项目为开源学习项目，不用于商业用途
